🧠 Task 3: State Change Detection and Save Logic

🎯 Task Objective

Detect meaningful changes in the input field and only enable the save button when the content has been actually modified. Re-entering the same content (e.g., deleting and retyping) should not trigger a save.

🛠️ Behavior

  • Start typing in the text box.
  • The Save button will be enabled only if the content is different from the last saved version.
  • Trivial changes (like adding/removing then retyping the same characters) won't trigger a save.
